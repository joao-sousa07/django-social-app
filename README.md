# Django Social App

Aplicação web desenvolvida com Django, Docker e MySQL.

## 📌 Tecnologias utilizadas

- Python
- Django
- MySQL
- Docker
- Docker Compose
- Bootstrap
- Git + GitHub

---

## 🚀 Funcionalidades

- Página principal com feed de posts
- Página de detalhe do post
- Página de posts por utilizador
- Sistema de likes
- Interface responsiva com Bootstrap
- Base de dados MySQL
